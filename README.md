## Stack

Todos los skills están construidos sobre **OpenClaw** (MIT) y son compatibles con cualquier instancia del agente. Diseñados para:

- 🇦🇷 Integraciones locales: ARCA/AFIP, MercadoPago, Tiendanube, WhatsApp Business API
- 🕐 Zona horaria ART (GMT-3) con formato de fecha DD/MM/AAAA
- 🇪🇸 Interfaz y documentación en español
- 🔒 Sin APIs externas en skills gratuitos — operan offline como base de conocimiento
